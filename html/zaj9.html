Poznajemy różne API

Dzisiejsze zajęcia będą bardziej praktyczne i będą bazowały na wiedzy z poprzednich. Będziemy rozwiązywać zadania dotyczące różnych API

The Cat API - https://thecatapi.com/

Zadanie 1.

Po ostatnich zajęciach powinna być już gotowa strona wyświetająca linki do wszystkich kwasów z KWAS API, dodaj do niej możliwość usuwania wybranych kwasów, można to zrobić na wiele różnych sposobów! Przykład:

https://catfact.ninja/#/
https://thecatapi.com/docs.html

Zadanie 2.

Wykorzystaj moc jaką dają ci CatFactsAPI oraz TheCatAPI i stwórz stronę która wyświetli zdjęcia losowego kota z dodatkiem losowej ciekawostki!

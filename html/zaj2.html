Warsztaty Webowe grupa średniozaawansowana

Przydatne wtyczki do Brackets'a

  • EMMET - pozwala na używanie skrótów usprawniających pracę np:
    • nazwa znacznika + tabulator - tworzy nam znacznik (razem z zamknięciem)
    • kombinacja typu: ul>li*3 + tabulator, sprawi, że wewnątrz elementu ul pojawi się element li i to w 3 kopiach. Można tak oczywiście robić z dowolnymi znacznikami
    • kombinacja znak + między nazwami znaczników + tabulator stworzy nam 2 znaczniki obok siebie (można grupować elementy za pomocą nawiasów)
    • znacznik{treść znacznika} - nawiasy klamrowe od razu wypełnią znaczniki treścią
    • lorem + tabulator - stworzy nam fragment losowego tekstu, jeśli po lorem dodamy cyfrę, wtedy mamy tekst złożony z tylu wyrazów ile wynosi liczba
    • znacznik#nazwa_id + tabulator - doda nam znacznik razem z id o wartości nazwa_id (z kropką to samo z klasami)
    • ! + tabulator - tworzy nasz szablon dokumentu html
    • c + tabulator - tworzy komentarz
    • resztę dostępnych skrótów można znaleźć tutaj
  • Custom work - dzięki niemu pliki wyświetlają się nad stroną, co ułatwia przechodzenie między nimi, dodaje też ikonki koło plików co ułatwia troszkę ich identyfikację
  • Beautify - porządkuje wcięcia i sprawia, że kod staje się czytelny
  • Brackets CSS Class Code hint - podpowiada nam jakie klasy i id posiadamy w dołączonym pliku css podczas pisania class=""
